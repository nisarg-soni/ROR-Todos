# ROR-Todos
A simple app to manage your To-Do list.

# Technologies used :

  - Ruby
  - Rails
  - Mysql
  
# Installation :
    - Create a empty Mysql database named 'rortodos'.
    - Clone the repo and cd into the project folder.

```sh
$ bundle install
$ rails db:migrate
$ rails s
```
    - Go to http://localhost:3000/ to check out the app. 
## Versions :
* **Ruby** : 2.4.1
* **Rails** : 5.1
